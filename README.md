# 仿去哪儿网Vue实现

## 技术栈
* Vue-cli 3.0脚手架工具
* Vue 2.5.17
* stylus
* Vue-Router
* Vuex
* Easy-Mock

## 项目安装
```
npm install
```

### 运行DevServer
```
npm run serve
```

### 打包编译
```
npm run build
```

### ESLint检查
```
npm run lint
```
